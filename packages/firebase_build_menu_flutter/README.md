# tekartik_firebase_build_menu_flutter

Flutter build utils in a menu

## Setup

In `pubspec.yaml`:

```yaml
dev_dependencies:
  tekartik_firebase_build_menu_flutter:
    git: 
        url: https://github.com/tekartik/build_flutter.dart
        ref: dart3a
        path: packages/firebase_build_menu_flutter
    version: '>=0.3.0'
```
